# WordPress Portfolio Theme

Custom WordPress theme built from scratch as a developer portfolio project.

## Tech Stack

- WordPress
- PHP
- SCSS
- Vite
- JavaScript / TypeScript
- Docker

## Features

- Custom WordPress theme architecture
- Modular SCSS structure
- Vite development workflow with Hot Module Replacement
- Responsive design
- Custom navigation and animations
- Dynamic content management (in progress)

## Development

The theme is developed locally using Docker and Vite.

## Roadmap
